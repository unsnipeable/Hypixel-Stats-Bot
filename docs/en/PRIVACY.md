# Cherry Privacy Policy

* Last Updated: April 30, 2026

This Privacy Policy ("Policy") describes how Cherry ("Service") collects and uses user information.

## 1. Information We Collect

The Service may collect the following information:

- Discord User ID
- Command usage history
- Minecraft in-game name (IGN)
- Other information necessary to provide the Service

## 2. Purpose of Use

Collected information is used for the following purposes:

- To provide and operate the Service
- To identify and authenticate users
- To prevent and respond to misuse
- To improve and develop the Service

## 3. Third-Party Services

The Service integrates with the following third-party services:

- Mojang API
- Hypixel API
- Urchin API

Information such as Minecraft player names may be sent to these services when necessary.

## 4. Data Storage and Management

- Collected information is retained only as long as necessary
- The operator takes appropriate measures to prevent data breaches, loss, or tampering

## 5. Sharing with Third Parties

User information will not be shared with third parties except in the following cases:

- When required by law
- When user consent has been obtained

## 6. User Rights

Users have the following rights regarding their data:

- Request access to their data
- Request correction or deletion of their data

Please contact the developer for such requests.

## 7. Minors

Users under the age of majority must obtain consent from a parent or guardian before using the Service.

## 8. Changes to This Policy

This Policy may be updated as necessary. Changes take effect upon publication.
