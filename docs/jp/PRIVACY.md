# Cherry プライバシーポリシー（Privacy Policy）

* 最終更新日：2026年4月30日

本プライバシーポリシー（以下、「本ポリシー」）は、Cherry（以下、「本サービス」）におけるユーザー情報の取扱いについて定めるものです。

## 第1条（取得する情報）

本サービスは、以下の情報を取得する場合があります。

* Discord User ID
* コマンド実行履歴
* Minecraft IGN
* その他、本サービスの機能提供に必要な情報

## 第2条（利用目的）

取得した情報は、以下の目的のために利用されます。

* 本サービスの機能提供および動作のため
* ユーザー認証および識別のため
* 不正利用の防止および対応のため
* サービス改善および開発のため

## 第3条（外部サービスとの連携）

本サービスは、以下の外部サービスと連携します。

* Mojang API
* Hypixel API
* Urchin API

これらのサービスに対して、必要に応じてMinecraftプレイヤー名などの情報が送信される場合があります。

## 第4条（情報の保存および管理）

* 取得した情報は、必要な期間のみ保存されます。
* 運営者は、情報の漏洩・紛失・改ざんを防ぐために適切な管理を行います。

## 第5条（第三者提供）

運営者は、以下の場合を除き、ユーザー情報を第三者に提供することはありません。

* 法令に基づく場合
* ユーザーの同意がある場合

## 第6条（ユーザーの権利）

ユーザーは、自身の情報について以下の権利を有します。

* 情報の開示請求
* 情報の訂正または削除の請求

これらの請求は、開発者までお問い合わせください。

## 第7条（未成年の利用）

未成年のユーザーは、保護者の同意を得た上で本サービスを利用するものとします。

## 第8条（ポリシーの変更）

本ポリシーは、必要に応じて変更されることがあります。変更後のポリシーは公開時点で効力を持ちます。

## 第9条（お問い合わせ）

本ポリシーに関するお問い合わせは、開発者 (@unsnipeable 又は @mtnkwawa) までご連絡ください。
